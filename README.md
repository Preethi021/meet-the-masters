# 🎓 Meet the Masters — Event & Mentorship Platform

A platform connecting aspiring professionals with industry experts 
through masterclasses, webinars, and 1-on-1 mentorship sessions.

## 🚀 Features
- One-click session registration with capacity and waitlist management
- Role-based protected flows: Admin · Speaker · Participant (React Router)
- Speaker approval and event management dashboard for admins
- Cross-platform mobile app (iOS & Android) built with Flutter
